## üö¶ Decision Framework: Should You Build X?

**Before building ANY feature, ask:**

1. **Do customers explicitly ask for it?** (Not "would be nice" - actual request)
2. **Will it help close a sale?** (Is it blocking revenue?)
3. **Can we charge more for it?** (Does it increase LTV?)
4. **Is it faster to buy than build?** (Can you use an API/service?)

**If answer to all 4 is NO ‚Üí Don't build it yet**
